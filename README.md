# Computational Drug Discovery

Machine learning and cheminformatics workflows for computational drug discovery using ChEMBL bioactivity data.

## Project Overview

This repository contains a series of bioinformatics and cheminformatics notebooks focused on:
- Bioactivity data preprocessing
- Molecular descriptor generation
- Feature engineering
- Regression and classification modeling
- Molecular solubility prediction
- Model comparison and interpretation

The project demonstrates the application of machine learning techniques for drug discovery and molecular property prediction.

**Key Workflows**
1. Bioactivity Modeling
 - ChEMBL bioactivity preprocessing
 - Molecular fingerprint generation
 - Random Forest regression
 - Model comparison and evaluation
2. Molecular Solubility Prediction
 - RDKit molecular descriptor calculation
 - Linear regression modeling
 - PyCaret automated machine learning workflows
 - SHAP-based model interpretation

Results

The models demonstrate the application of machine learning techniques for predicting molecular properties and analyzing bioactivity-related datasets.
