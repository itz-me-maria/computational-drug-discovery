
# Notebooks Directory

This directory contains Jupyter notebooks covering the complete computational drug discovery and cheminformatics workflow.

## Notebook Overview

| Notebook | Description |
|---|---|
| 01_bioactivity_data_preprocessing.ipynb | Bioactivity data collection and preprocessing |
| 02_exploratory_data_analysis.ipynb | Exploratory data analysis and statistical visualization |
| 03_descriptor_calculation_and_dataset_preparation.ipynb | Molecular descriptor generation and dataset preparation |
| 04_random_forest_regression_model.ipynb | Random Forest regression modeling |
| 05_regression_model_comparison.ipynb | Comparison of multiple regression algorithms |
| 06_molecular_solubility_prediction.ipynb | Molecular solubility prediction using linear regression |
| 07_solubility_prediction_pycaret.ipynb | Automated regression modeling using PyCaret |
| 08_solubility_prediction_pycaret.ipynb | Advanced PyCaret workflow and model interpretation |

The notebooks demonstrate machine learning applications for computational drug discovery, molecular property prediction, and cheminformatics analysis.
